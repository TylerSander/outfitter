Outfitter is a curated app store for your new (or newly reformatted) computer. It installs everything through your operating system's own trusted package manager — winget on Windows, Homebrew on Mac, Flatpak on Linux — so nothing ever comes from a mystery download site. Outfitter never hosts installers; it just knows where the real ones live.
